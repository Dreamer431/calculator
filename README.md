## 功能特点

### 基础运算
- 支持加减乘除四则运算（左结合）
- 支持幂运算（使用 `^` 符号，右结合，如 `2^3^2 = 512`）
- 支持括号嵌套
- 支持小数计算
- 支持数字与括号之间的隐式乘法（如 `2(3+4)`, `(2)(3)`, `2pi`）
- 支持科学计数法（如 `1.23e-4`）

### 三角函数
- `sin(x)`：正弦函数
- `cos(x)`：余弦函数
- `tan(x)`：正切函数
- `asin(x)`：反正弦函数
- `acos(x)`：反余弦函数
- `atan(x)`：反正切函数
- 支持角度模式和弧度模式切换

### 其他数学函数
- `sqrt(x)`：平方根函数
- `log(x)`：常用对数（以10为底）
- `ln(x)`：自然对数
- `abs(x)`：绝对值
- `rad(x)`：角度转弧度
- `deg(x)`：弧度转角度

### 常量支持
- `pi`：圆周率（3.14159265358979...）
- `e`：自然对数的底（2.71828182845905...）

### 错误处理
- 提供详细的错误提示和位置信息
- 括号匹配检查
- 数值范围检查
- 表达式格式验证
- 分类错误代码

### 用户体验
- 命令历史记录功能（最近5条）
- 详细的帮助信息
- 空格容忍（如 `1 + 1`）

## 编译要求

- 编译器：GCC（推荐 MinGW-w64 或 TDM-GCC）
- 构建工具：make（可选）

### 跨平台兼容性

项目支持以下平台：
- **Windows**：使用 MinGW-w64 或 TDM-GCC
- **Linux**：使用 GCC
- **macOS**：使用 Clang 或 GCC

## 编译方法

### 使用 Make（推荐）

```bash
# 编译主程序
make

# 编译并运行测试
make test

# 清理编译产物
make clean
```

### 手动编译

```bash
# 编译所有源文件
gcc -Wall -Wextra -O2 -Iinclude -Itest \
    src/core/*.c src/utils/*.c \
    -o calculator -lm

# 编译测试
gcc -Wall -Wextra -O2 -Iinclude -Itest \
    src/core/expression_evaluator.c \
    src/core/operator_handling.c \
    src/core/error_handling.c \
    src/utils/*.c \
    test/*.c \
    -o test_runner -lm
```

## 使用方法

1. 在命令行中运行程序：
   ```bash
   ./calculator
   ```

2. 程序启动后：
   - 输入数学表达式进行计算
   - 输入 `mode` 切换角度/弧度模式
   - 输入 `history` 查看历史记录
   - 输入 `help` 查看帮助信息
   - 输入 `q` 退出程序

### 示例

```
请输入计算表达式 [角度]: 2^3^2
2^3^2  = 512

请输入计算表达式 [角度]: sin(30) + cos(60)

请输入计算表达式 [角度]: 2pi
2pi  = 6.2831853072

请输入计算表达式 [角度]: sqrt(3^2 + 4^2)
sqrt(3^2 + 4^2)  = 5
```

## 表达式规则

### 运算符
| 运算符 | 说明 | 结合性 | 优先级 |
|--------|------|--------|--------|
| `+` `-` | 加减法 | 左结合 | 低 |
| `*` `/` | 乘除法 | 左结合 | 中 |
| `^` | 幂运算 | 右结合 | 高 |

### 函数
- 三角函数：`sin`, `cos`, `tan`, `asin`, `acos`, `atan`
- 对数函数：`log`（常用对数）、`ln`（自然对数）
- 其他函数：`sqrt`, `abs`, `rad`, `deg`

### 常量
- `pi`：圆周率
- `e`：自然对数的底

### 数字格式
- 整数：`123`
- 小数：`123.456`
- 科学计数法：`1.23e-4`、`1.23E+5`

### 隐式乘法
支持以下形式的隐式乘法：
- 数字后跟括号：`2(3+4)` = `2*(3+4)` = 14
- 括号后跟括号：`(2)(3)` = `(2)*(3)` = 6
- 数字后跟常量：`2pi` = `2*pi` ≈ 6.28

### 空格
表达式中可以包含空格，会被自动忽略：
- `1 + 1` = 2
- `sin( 90 )` = 1

## 注意事项

- 表达式长度不要超过99个字符（超过时程序会给出警告并自动截断）
- 括号嵌套不要超过100层
- 数值范围限制：
  - 最大值：约 1.7×10^308
  - 最小值：约 2.2×10^-308
  - 整数精度：使用 int64_t，支持约 ±9.2×10^18
